# Encapsulation Practice 02

## Goal

Practice encapsulation with validation logic.

## Task

Implement a `Person` class that stores a person's age.

## Requirements

The class must contain:

- a private `age` field

Implement:
setAge(int age)
getAge()


### Rules

- Age cannot be negative
- Age cannot be greater than 130
- If an invalid age is provided, the value should **not change**

## Important

Do not modify the tests.

Your goal is to implement the class so all tests pass.

## Learning Objectives

- Encapsulation
- Data validation
- Writing defensive setters
